Variables in Java

Review

In Java, variables are used to store data values that can be manipulated and referenced in a program. Here's an overview of variables in Java:

  1. Declaration: To use a variable in Java, it must first be declared with a specific data type and identifier (name). For example:
    int num;
String name;
double price;
  1. Data Types: Java supports various data types for variables, including:
    • Primitive Data Types: int, double, char, boolean, etc., which hold simple values.
    • Reference Data Types: String, arrays, classes, interfaces, etc., which hold references to objects.
  1. Initialization: Variables can be initialized at the time of declaration or later in the program. If not explicitly initialized, variables are assigned default values based on their data types.
    int num = 10;           // Initialized at declaration
String name = "Java";   // Initialized at declaration
double price;           // Declaration without initialization
price = 99.99;          // Initialized later in the program
  1. Scope: The scope of a variable defines where in the program it can be accessed. In Java, variables can have different scopes:
    • Local Variables: Defined within a method or block and are only accessible within that method or block.
    • Instance Variables (Fields): Defined within a class but outside any method, constructors, or blocks. They are accessible to all methods in the class.
    • Static Variables (Class Variables): Shared among all instances of a class. They are declared using the static keyword and are associated with the class rather than any specific instance.
  1. Naming Conventions: Variable names in Java should follow certain conventions:
    • Start with a letter, underscore (_), or dollar sign ($).
    • Subsequent characters can be letters, digits, underscores, or dollar signs.
    • Names are case-sensitive and should be meaningful and descriptive.
  1. Final Variables: Variables declared with the final keyword are constants and cannot be reassigned once initialized.
    final int MAX_VALUE = 100;
  1. Type Inference (Java 10 and later): Java supports type inference with the var keyword, allowing the compiler to infer the type of a variable based on its initialization.
    var message = "Hello, World!"; // Inferred as String

Variables play a crucial role in Java programs by storing and manipulating data, and understanding their usage is fundamental to writing efficient and maintainable code.
