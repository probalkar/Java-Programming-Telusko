Enum class in Java

Review

In Java, an enum class is a special type of class used to define a fixed set of constants. Enums are often used to represent a collection of related constants, making the code more readable, maintainable, and type-safe. Here's an explanation of enum classes in Java:

Declaration:

  • Enum classes are declared using the enum keyword followed by the enum name.
  • Inside an enum class, you list the constants, separated by commas, which represent the possible values of the enum.

Example of enum class declaration:

enum Day {
    MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY, SUNDAY;
}

Constants and Instances:

  • Each constant in an enum class is an instance of the enum class and is implicitly public, static, and final.
  • You can access enum constants using the enum class name followed by the constant name.

Example of accessing enum constants:

Day day = Day.MONDAY;

Methods and Fields:

  • Enum classes can have methods, constructors, and fields just like regular classes.
  • You can define custom methods and fields for enum constants to add behavior and additional information.

Example of enum class with methods and fields:

enum Day {
    MONDAY("Monday", 1), TUESDAY("Tuesday", 2), WEDNESDAY("Wednesday", 3), THURSDAY("Thursday", 4),
    FRIDAY("Friday", 5), SATURDAY("Saturday", 6), SUNDAY("Sunday", 7);

    private final String displayName;
    private final int dayNumber;

    Day(String displayName, int dayNumber) {
        this.displayName = displayName;
        this.dayNumber = dayNumber;
    }

    public String getDisplayName() {
        return displayName;
    }

    public int getDayNumber() {
        return dayNumber;
    }
}

Using Enum Constants:

  • Enums are typically used to represent a fixed set of constants, such as days of the week, colors, status codes, etc.
  • You can use enum constants in switch-case statements, if-else conditions, and as method parameters or return types.

Example of using enum constants in a switch-case statement:

Day day = Day.MONDAY;

switch (day) {
    case MONDAY:
        System.out.println("It's Monday.");
        break;
    case TUESDAY:
        System.out.println("It's Tuesday.");
        break;
    // Other cases...
}

Key Points:

  • Enum classes in Java provide a convenient way to represent a fixed set of constants.
  • Enum constants are instances of the enum class and are implicitly public, static, and final.
  • Enum classes can have methods, constructors, and fields, allowing you to add behavior and additional information to enum constants.
  • Enums are commonly used to improve code readability, type safety, and maintainability.

By using enum classes in Java, you can create more expressive and robust code that clearly defines and encapsulates a set of related constants.

enum Laptop{
    Macbook(2500), Zephyrus, Legion(1700), Razer(2300);

    private int price;

    private Laptop(){
        price = 1500;
    }

    private Laptop(int price){
        this.price = price;
    }

    public int getPrice() {
        return price;
    }
}

public class Enum {
    public static void main(String[] args) {
        Laptop mac = Laptop.Macbook;
        System.out.println(mac.getPrice());

        for(Laptop lap : Laptop.values()){
            System.out.println(lap + " : " + lap.getPrice());
        }
    }
}
