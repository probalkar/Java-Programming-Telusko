Inner class in Java

In Java, an inner class is a class defined within another class. Inner classes provide a way to logically group classes and improve encapsulation and code organization. There are several types of inner classes, including nested inner classes, local inner classes, anonymous inner classes, and static nested classes. Here's an explanation of each type:

1. Nested Inner Classes:

  • A nested inner class is declared inside another class as a member of that class.
  • It has access to all members of the enclosing class, including private members.
  • Nested inner classes can be static or non-static.
  • Non-static nested inner classes are also called inner classes.

Example of a nested inner class:

class Outer {
    private int outerVar;

    class Inner {
        void display() {
            System.out.println("OuterVar: " + outerVar);
        }
    }
}

2. Local Inner Classes:

  • A local inner class is declared within a method or a block of code.
  • It has access to local variables and parameters of the enclosing method or block, but these variables must be effectively final or final.
  • Local inner classes are not accessible outside the method or block in which they are defined.

Example of a local inner class:

class Outer {
    void display() {
        int localVar = 10;

        class Inner {
            void print() {
                System.out.println("LocalVar: " + localVar);
            }
        }

        Inner inner = new Inner();
        inner.print();
    }
}

3. Anonymous Inner Classes:

  • An anonymous inner class is a local inner class without a name that is declared and instantiated at the same time.
  • It is often used for implementing interfaces or extending classes in a concise manner.
  • Anonymous inner classes do not have constructors.

Example of an anonymous inner class:

interface Message {
    void display();
}

class Outer {
    Message message = new Message() {
        @Override
        public void display() {
            System.out.println("Anonymous inner class");
        }
    };
}

4. Static Nested Classes:

  • A static nested class is declared as a static member of another class.
  • It can access only static members of the enclosing class and does not have access to instance variables and methods.
  • Static nested classes are accessed using the enclosing class name.

Example of a static nested class:

class Outer {
    static class Nested {
        void print() {
            System.out.println("Static nested class");
        }
    }
}

Key Points:

  • Inner classes allow for better encapsulation, code organization, and logical grouping of classes.
  • Nested inner classes, local inner classes, anonymous inner classes, and static nested classes each have their own use cases and characteristics.
  • Inner classes can access members of the enclosing class, including private members, which can be useful for implementing encapsulation.

Inner classes are a powerful feature of Java that enable developers to write more modular and readable code by organizing related classes together and enhancing encapsulation and code reusability.

class A{
    public void show(){
        System.out.println("in show");
    }

    class B{    // non-static inner class
        public void config(){
            System.out.println("in config");
        }
    }

    static class C{ // static inner class
        public void print(){
            System.out.println("in C print");
        }
    }
}

public class InnerClass {
    public static void main(String[] args) {
        A obj = new A();
        obj.show();

        A.B obj1 = obj.new B(); // need object of outer class to create object of inner class if the inner class is not static
        obj1.config();

        A.C obj2 = new A.C();   // need to follow this syntax if the inner class is static and want to create an object of inner class
        obj2.print();
    }
}
