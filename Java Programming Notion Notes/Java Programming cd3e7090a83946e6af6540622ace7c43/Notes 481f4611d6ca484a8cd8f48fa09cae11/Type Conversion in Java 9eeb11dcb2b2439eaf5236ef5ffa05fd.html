Type Conversion in Java

In Java, type conversion refers to the process of converting a value from one data type to another. There are two main types of type conversion: implicit conversion (also known as widening conversion) and explicit conversion (also known as narrowing conversion or casting).

  1. Implicit Conversion (Widening):
    • Implicit conversion occurs when the data type of a value is automatically converted to a larger data type without the need for explicit casting.
    • It happens when there is no loss of precision or information in the conversion.
    • For example, converting an integer to a floating-point number or promoting a smaller data type to a larger data type.
    • Example:
      int numInt = 10;
double numDouble = numInt; // Implicit conversion from int to double
  1. Explicit Conversion (Narrowing or Casting):
    • Explicit conversion occurs when the data type of a value is manually converted to a smaller data type using casting.
    • It is necessary when converting from a larger data type to a smaller data type, potentially resulting in loss of precision or information.
    • Casting is performed by specifying the desired data type in parentheses before the value to be converted.
    • Example:
      double numDouble = 3.14;
int numInt = (int) numDouble; // Explicit conversion (casting) from double to int
    • Note that explicit conversion may result in data loss or truncation if the value being converted cannot be represented accurately in the target data type.
  1. Type Promotion and Demotion:
    • Type promotion refers to the automatic conversion of operands to a common data type in expressions or operations.
    • Type demotion refers to the manual conversion of a data type to a narrower type using casting.
    • Java follows specific rules for type promotion and demotion to ensure that data is converted safely and accurately.
  1. String Conversion:
    • Java also supports converting other data types to and from strings using methods like toString() and valueOf() provided by wrapper classes or using string concatenation.
    • Example:
      int numInt = 10;
String numString = Integer.toString(numInt); // Convert int to String

Type conversion is an essential aspect of Java programming, allowing developers to work with different data types and perform operations effectively while ensuring data integrity and accuracy.

public class Type_Conversion {
    public static void main(String[] args) {
        // Implicit type casting
        byte b = 12;
        int a = b;
        System.out.println(a);  // works perfectly

        // int c = 12;
        // byte d = c;     // this will throw an error due to the fear of losing information as byte is smaller than int

        // explicit type casting
        int c = 12;
        byte d = (byte)c;
        System.out.println(d);  // this will work

        // when value out of range
        int e = 257;
        byte f = (byte)e;   // behind the scene computation: f = e%256 <- 256 is total range of byte datatype => f = 1
        System.out.println(f);

        float g = 5.6f;
        int h = (int)g;     // we will lose .6 here
        System.out.println(h);

        // type promotion
        byte b1 = 10;
        byte b2 = 30;
        int result = b1 * b2;   // b1*b2 will result a number out of range of byte datatype so by storing it in int datatype compiler is promoting it's datatype
        System.out.println(result);
    }
}
